# SubscriptionManager

Данное мобильное приложение разработано для систематизации коллекции подписок. Приложение предназначено для пользователей, которые хотят отслеживать свои финансовые обязательства по различным подпискам и сервисам.

## Использование
Для работы с приложением необходимо следующее программное обеспечение:
* Устройство под управлением Android 8.0 или выше
* Эмулятор Android в среде разработки Android Studio

Перед началом работы с приложением необходимо выполнить следующие действия:
* Запустите приложение "SubscriptionManager" на устройстве.
* Если вы новый пользователь, нажмите "Зарегистрироваться".
* Заполните форму регистрации (Имя пользователя, Пароль, Подтверждение пароля) и нажмите "Зарегистрироваться".
* После успешной регистрации войдите в систему, используя свои учетные данные.
* На главном экране нажмите "Добавить подписку", чтобы начать формировать свой список подписок.

### Добавление подписки
1. Введите название подписки в поле "Название подписки"
2. Введите стоимость в поле "Цена"
3. Введите категорию в поле "Категория"
4. Выберите дату начала, нажав на поле "Дата начала"
5. Выберите дату окончания, нажав на поле "Дата окончания"
6. Нажмите "Сохранить"

### Управление подписками в списке
* **Просмотр подписок:** Все подписки отображаются в виде списка с названием, категорией, ценой и периодом действия
* **Удаление подписки:** Нажмите кнопку "Удалить" на карточке подписки и подтвердите действие
* **Выход из аккаунта:** Нажмите кнопку "Выйти" внизу главного экрана

Пояснение по тестовым путям:
* Т1: Запуск приложения → Открытие окна авторизации → Успешный вход в систему → Отображение главного окна → Закрытие приложения. (Базовый сценарий: вход и сразу выход.)
* Т2: Запуск приложения → Окно авторизации → Успешный вход → Главное окно → Переход в окно добавления подписки → Возврат в главное окно → Закрытие приложения. (Проверка открытия и закрытия окна добавления подписки без сохранения.)
* Т3: Запуск приложения → Окно авторизации → Переход в окно регистрации → Создание нового аккаунта → Возврат к окну авторизации → Успешный вход → Главное окно → Закрытие приложения. (Сценарий регистрации нового пользователя с последующим входом.)
* Т4: Запуск приложения → Окно авторизации → Успешный вход → Главное окно → Окно добавления подписки → Возврат в главное окно → Выход из аккаунта (возврат к авторизации) → Закрытие приложения. (Проверка добавления подписки и выхода из системы.)
* Т5: Запуск приложения → Окно авторизации → Переход в регистрацию → Создание аккаунта → Возврат к авторизации → Успешный вход → Главное окно → Окно добавления подписки → Возврат в главное окно → Закрытие приложения. (Комбинированный сценарий: регистрация → вход → добавление подписки.)
* Т6: Запуск приложения → Окно авторизации → Успешный вход → Главное окно → Окно добавления подписки → Возврат в главное окно → Повторный переход в окно добавления подписки → Возврат в главное окно → Закрытие приложения. (Проверка многократного открытия/закрытия окна добавления подписки.)
* Т7: Запуск приложения → Окно авторизации → Успешный вход → Главное окно → Выход из аккаунта (возврат к авторизации) → Закрытие приложения. (Альтернативный сценарий: вход, затем выход без дополнительных действий.)

## FAQ
### Где найти кнопку для добавления подписки?
* Она находится в нижней части главного экрана и имеет надпись "Добавить подписку".

### Как удалить подписку из списка?
* Нужно нажать кнопку "Удалить" на карточке подписки и подтвердить действие в диалоговом окне.

### Как изменить данные подписки?
* В текущей версии изменение подписки не предусмотрено — удалите существующую и создайте новую.

### Почему дата окончания не может быть раньше даты начала?
* Приложение автоматически проверяет корректность дат и не позволяет сохранить подписку с некорректным периодом действия.
